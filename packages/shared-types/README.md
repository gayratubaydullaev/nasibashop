# @nasibashop/shared-types

Общие TypeScript-типы для фронтендов (согласованы с JSON **product-service**).

## Использование

Из приложения в монорепо:

```json
"dependencies": {
  "@nasibashop/shared-types": "file:../../packages/shared-types"
}
```

## Скрипты

```bash
npm ci
npm run typecheck
```
